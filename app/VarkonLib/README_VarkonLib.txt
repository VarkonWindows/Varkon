Do no remember how to build mbo files for the directory Varkon/lib
Make command files do not work for Windows

1. Copy mbs files from 
   Varkon/sources/Vlib/Analysis
   Varkon/sources/Vlib/dxf
   Varkon/sources/Vlib/edit
   Varkon/sources/Vlib/iges
   Varkon/sources/Vlib/stl
   
   to Varkon project VarkonLib (or any dummy project)
   Varkon/app/mbs
   
2. Compile all mbs code
   Varkon menu mbs .> part -> compile all
   (There were errors, some failed)
   
3. Copy mbo files to Varkon lib
   mbo files from Varkon/app/lib
   to Varkon/lib
